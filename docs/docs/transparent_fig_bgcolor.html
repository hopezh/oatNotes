Transparent bgcolor for figure

Use the rgba() attrib to set the alpha, or transparency, of the plot/paper components of a figure.

without transparency 

import plotly.express as px

df = px.data.iris()

fig = px.scatter(
    df,
    x="sepal_width",
    y="sepal_length",
    color="species",
    size='petal_length',
    hover_data=['petal_width']
)

# fig.update_layout({
#     'plot_bgcolor' : 'rgba(255, 255, 0,   0.5)',
#     'paper_bgcolor': 'rgba(0,   0,   255, 0.5)',
# })

fig.show()

with transparency 

import plotly.express as px

df = px.data.iris()

fig = px.scatter(
    df,
    x="sepal_width",
    y="sepal_length",
    color="species",
    size='petal_length',
    hover_data=['petal_width']
)

fig.update_layout({
    'plot_bgcolor' : 'rgba(255, 255, 0,   0.5)',
    'paper_bgcolor': 'rgba(0,   0,   255, 0.5)',
})

fig.show()

Use the `rgba()` attrib to set the `alpha`, or `transparency`, of the `plot`/`paper` components of a `figure`.

::: {.columns .column-screen .column-screen-insert}

::: column

<h2 style="text-align: center;">without transparency</h2>

```{python}
import plotly.express as px

df = px.data.iris()

fig = px.scatter(
    df,
    x="sepal_width",
    y="sepal_length",
    color="species",
    size='petal_length',
    hover_data=['petal_width']
)

# fig.update_layout({
#     'plot_bgcolor' : 'rgba(255, 255, 0,   0.5)',
#     'paper_bgcolor': 'rgba(0,   0,   255, 0.5)',
# })

fig.show()
```

:::

::: column

<h2 style="text-align: center;">with transparency</h2>

```{python}
#| source-line-numbers: "14-17"
import plotly.express as px

df = px.data.iris()

fig = px.scatter(
    df,
    x="sepal_width",
    y="sepal_length",
    color="species",
    size='petal_length',
    hover_data=['petal_width']
)

fig.update_layout({
    'plot_bgcolor' : 'rgba(255, 255, 0,   0.5)',
    'paper_bgcolor': 'rgba(0,   0,   255, 0.5)',
})

fig.show()
```
