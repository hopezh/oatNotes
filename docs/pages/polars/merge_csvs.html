# (lazy-)filter a DF by a list of items in a column

```python
%%time

strings_to_filter = [
    '1-4-NB05-10',
    '1-4-NB05-9',
    '1-8-NB02-4',
]

query = (                               # <2>
    pl.scan_csv(csv_path_glob)
      .filter(pl.col('string_name')
      .is_in(strings_to_filter))
)

df_filtered = query.collect()           # <3>

df_filtered
```
